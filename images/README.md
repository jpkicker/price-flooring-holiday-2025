## Image Requirements

**Format:** JPG or WebP
**Minimum Size:** 800px wide x 560px tall
**Recommended:** 1200px+ wide for high-resolution displays
**Target File Size:** Under 200KB per image (compress if needed)

**Style Guidelines:**
- Lifestyle images showing installed flooring
- Well-lit, professional photography
- Contemporary room settings
- Should showcase the flooring prominently

## Image Optimization (Optional)

To reduce file size and improve page load speed:

1. **Compress images:** Use TinyPNG (https://tinypng.com)
2. **Resize if needed:** Max 1920px wide is plenty
3. **Convert to WebP:** For even smaller file sizes (optional)

**Compression reduces file size by 50-70% without visible quality loss**
